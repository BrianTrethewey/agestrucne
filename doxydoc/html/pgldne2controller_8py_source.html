101  This list of parameters are presumably the common settings, used
102  by this object when it makes a configuration file that
103  the executable reads in when it gets an arg of the form,
104  "c:<configfile>" in the command command. These comments
105  head the source function RunMultiCommon, called when main()
106  parses the args noted above:
107  // 1. Number of (positive) critical values
108  // 2. Critical values if applicable (i.e., number on line 2 is positive)
109  // 3. Plan/Generations for Temporal if applicable.
110  // 4. Tabular-format output file(s)

112  // 6. Random/Monogamy for LD if applicable

114  // 8. Population range
115  // 9. Locus ranges to run
116  // 10. Common output file name
117  // Then the rest, at each line, are input file names including paths
118  // if necessary.

120  Trials show that line 3 turns tab files on/off, and

123  when lines 1 and 2 are standard number of crits and
124  matching number of entries for crits, is skipped.
125  We adjust our dict accordingly.
